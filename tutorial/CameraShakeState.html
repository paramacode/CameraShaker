CameraShakeState

An enum that represents the lifecycle state of a CameraShakeInstance. Each instance transitions through these states as it fades in, sustains, fades out, and becomes inactive.

Values​

NameValueDescription
FadingIn0The shake is ramping up to full magnitude.
FadingOut1The shake is fading down to zero.
Sustained2The shake is at full magnitude and looping until explicitly stopped.
Inactive3The shake has finished or been stopped. If deleteOnInactive is true, it will be removed automatically.

State flow​

FadingIn → Sustained → FadingOut → Inactive
    │                                  ▲
    └──────────────────────────────────┘
         (if sustain = false)
  • One-shot shakes (shakeOnce): FadingIn → FadingOut → Inactive
  • Sustained shakes (shakeSustained): FadingIn → Sustained → FadingOut → Inactive
  • Calling .stop() on any instance immediately sets it to Inactive.

Checking state​

  • TypeScript
  • Luau
import { CameraShaker, CameraShakeState, Presets } from "@rbxts/camerashaker";

const shaker = new CameraShaker();
shaker.start();

const shake = shaker.shake(Presets.Explosion());

if (shake.state === CameraShakeState.FadingIn) {
    print("Shake is fading in...");
}

if (shake.state === CameraShakeState.Inactive) {
    print("Shake has finished.");
}
local CameraShakerModule = require(path.to.CameraShaker)
local CameraShaker = CameraShakerModule.CameraShaker
local CameraShakeState = CameraShakerModule.CameraShakeState
local Presets = CameraShakerModule.Presets

local shaker = CameraShaker.new()
shaker:start()

local shake = shaker:shake(Presets.Explosion())

if shake.state == CameraShakeState.FadingIn then
    print("Shake is fading in...")
end

if shake.state == CameraShakeState.Inactive then
    print("Shake has finished.")
end
