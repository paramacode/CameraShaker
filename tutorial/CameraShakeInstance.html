CameraShakeInstance

Represents a single shake effect. Each instance holds its own magnitude, roughness, fade timings, and influence values. Instances are added to a CameraShaker which combines them every frame.

new CameraShakeInstance(options)​

ParameterTypeDescription
optionsCameraShakeOptionsConfiguration object for the shake.

CameraShakeOptions​

FieldTypeDefaultDescription
magnitudenumber-The intensity of the shake. Must be >= 0.
roughnessnumber-The frequency or speed of the shake. Must be >= 0.
fadeInDurationnumber0Time in seconds to reach full magnitude.
fadeOutDurationnumber0Time in seconds to fade out to zero.
positionInfluenceVector3(0.15, 0.15, 0.15)How much the shake affects the camera's position (X, Y, Z).
rotationInfluenceVector3(1, 1, 1)How much the shake affects the camera's rotation (Pitch, Yaw, Roll).
zoomIntensitynumber0Intensity of the FOV zoom effect. Must be >= 0.
zoomSpeednumber5Speed of the FOV zoom oscillation. Must be > 0.
deleteOnInactivebooleantrueIf true, the instance is automatically removed from the shaker when it becomes inactive.
info

magnitude and roughness are required. All other fields are optional and fall back to their defaults.

Properties​

PropertyTypeDescription
magnitudenumberCurrent shake intensity.
roughnessnumberCurrent shake frequency / speed.
positionInfluenceVector3Position influence multiplier.
rotationInfluenceVector3Rotation influence multiplier.
fadeInDurationnumberFade-in time in seconds.
fadeOutDurationnumberFade-out time in seconds.
currentFadeTimenumberInternal fade progress (0 to 1).
roughnessModifiernumberMultiplier applied to roughness. Default 1.
magnitudeModifiernumberMultiplier applied to magnitude. Default 1.
sustainbooleanWhether the shake loops after fading in.
stateCameraShakeStateThe current state of the instance.
deleteOnInactivebooleanWhether the instance auto-removes when inactive.
zoomIntensitynumberFOV zoom intensity.
zoomSpeednumberFOV zoom oscillation speed.

.startFadeIn(fadeTime?)​

ParameterTypeDefaultDescription
fadeTimenumberCurrent fadeInDurationOverride the fade-in time in seconds.

Sets the instance state to FadingIn and marks it as sustained. If fadeTime is provided, it updates fadeInDuration.

.startFadeOut(fadeTime?)​

ParameterTypeDefaultDescription
fadeTimenumberCurrent fadeOutDurationOverride the fade-out time in seconds.

Sets the instance state to FadingOut. If fadeTime is provided, it updates fadeOutDuration.

.stop()​

Immediately stops the shake by setting its state to Inactive and resetting currentFadeTime to 0.

.update(deltaTime)​

ParameterTypeDescription
deltaTimenumberTime elapsed since the last frame.

Returns: (Vector3, Vector3, number) — position offset, rotation offset, and zoom value.

caution

This method is called internally by CameraShaker every frame. You typically do not need to call it manually.

Creating a custom shake​

  • TypeScript
  • Luau
import { CameraShakeInstance } from "@rbxts/camerashaker";

const shake = new CameraShakeInstance({
    magnitude: 3,
    roughness: 12,
    fadeInDuration: 0.1,
    fadeOutDuration: 0.5,
    positionInfluence: new Vector3(0.25, 0.25, 0.25),
    rotationInfluence: new Vector3(2, 2, 2),
    zoomIntensity: 1.5,
    zoomSpeed: 8,
});
local CameraShakerModule = require(path.to.CameraShaker)
local CameraShakeInstance = CameraShakerModule.CameraShakeInstance

local shake = CameraShakeInstance.new({
    magnitude = 3,
    roughness = 12,
    fadeInDuration = 0.1,
    fadeOutDuration = 0.5,
    positionInfluence = Vector3.new(0.25, 0.25, 0.25),
    rotationInfluence = Vector3.new(2, 2, 2),
    zoomIntensity = 1.5,
    zoomSpeed = 8,
})

Modifying at runtime​

You can tweak an instance's properties while it's active:

  • TypeScript
  • Luau
const shake = shaker.shakeOnce(2, 10, 0.1, 1);

// Double the intensity mid-shake
shake.magnitudeModifier = 2;

// Slow down the roughness
shake.roughnessModifier = 0.5;
local shake = shaker:shakeOnce(2, 10, 0.1, 1)

-- Double the intensity mid-shake
shake.magnitudeModifier = 2

-- Slow down the roughness
shake.roughnessModifier = 0.5

Controlling fade manually​

  • TypeScript
  • Luau
const shake = new CameraShakeInstance({
    magnitude: 2,
    roughness: 8,
    fadeInDuration: 1,
    fadeOutDuration: 2,
});

// Start sustained shake
shaker.shakeSustained(shake);

// Manually trigger fade-out with a custom time
task.delay(3, () => {
    shake.startFadeOut(0.5);
});
local CameraShakerModule = require(path.to.CameraShaker)
local CameraShakeInstance = CameraShakerModule.CameraShakeInstance

local shake = CameraShakeInstance.new({
    magnitude = 2,
    roughness = 8,
    fadeInDuration = 1,
    fadeOutDuration = 2,
})

-- Start sustained shake
shaker:shakeSustained(shake)

-- Manually trigger fade-out with a custom time
task.delay(3, function()
    shake:startFadeOut(0.5)
end)
